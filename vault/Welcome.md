# Welcome to Vitra

This is your new **Obsidian-compatible** notes app.

## Features

- [[Another Note]] - WikiLinks work!
- #tag support
- Full-text search
- Live preview while editing

## Getting Started

1. Create a new note from the sidebar
2. Use `[[Note Name]]` to link between notes
3. Use `#tags` to organize
